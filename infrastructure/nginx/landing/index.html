├── Schema: authelia — SSO 统一认证 (自动管理)
├── Schema: wms     — WMS 操作数据 (12表 + 4视图)
├── Schema: oms     — OMS 订单管理 (4表 + 2视图)
├── Schema: wf      — n8n 工作流元数据 (自动管理)
├── Schema: bi      — BI 引擎元数据 (自动管理)
└── Schema: ai      — AI 向量数据 (pgvector, 预留)

开发者体验

5 分钟，从零构建您的数字平台

Docker Compose 一键拉起，全平台就绪

bash 
# 1. 克隆仓库
git clone https://github.com/foodsaid/dp
cd Digital-Platform

# 2. 配置环境 (⚠️ 必须设置所有密码字段!)
cp .env.example .env

# 3. 一键启动 (自动检测平台 + 创建网络 + 权限预检)
bash scripts/dev-up.sh

# 4. 访问平台 🎉
WMS      →  http://localhost:<DP_GATEWAY_PORT>/wms/
OMS      →  http://localhost:<DP_GATEWAY_PORT>/wms/oms.html
BI       →  http://localhost:<DP_GATEWAY_PORT>/bi
Workflow →  http://localhost:<DP_WF_PORT>
